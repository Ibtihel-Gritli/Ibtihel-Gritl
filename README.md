## 👩‍💻 À Propos de Moi

Je suis **Gritli Ibtihel**, étudiante en **3ème année Licence GLSI** (Génie Logiciel & Systèmes d'Information) à la **Faculté des Sciences de Monastir, Tunisie**.

Je développe des **applications web full-stack** intégrant des technologies modernes, avec un focus particulier sur l'**intégration de l'IA**, le **code propre** et les **bonnes pratiques de développement**.

- 🌐 **Frontend** : HTML, CSS, JavaScript, PHP, Next.js
- 🗄️ **Bases de données** : MySQL, Oracle, H2, JPA
- 🏗️ **Architecture** : UML, MVC, REST API, Agile/Scrum
- 🤖 **IA intégrée** : Groq API (Llama 3.3-70B), WebSocket, STOMP
